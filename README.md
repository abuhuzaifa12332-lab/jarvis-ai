# 🤖 Jarvis AI — Futuristic Virtual Voice Assistant (V3)

An interactive, web-based AI voice assistant inspired by Jarvis from Iron Man. Built using HTML, CSS, JavaScript, and the Web Speech API, Jarvis AI can listen to voice commands, respond using speech synthesis, open websites, perform searches, and provide useful information directly inside the browser.

## 🚀 Project Overview

**Jarvis AI (V3)** is a browser-based virtual assistant designed to provide a hands-free interactive experience through voice commands. The application utilizes modern Web APIs to convert spoken words into actions and generate spoken responses in real time.

Inspired by Tony Stark's intelligent assistant, this project demonstrates how powerful browser technologies can be combined to create an engaging voice-controlled user experience without requiring backend infrastructure.

### Why Jarvis AI?

* Fully client-side architecture
* No server-side processing required
* Real-time speech recognition
* Natural voice responses
* Fast and lightweight
* Responsive futuristic user interface
* Easy deployment through GitHub Pages

## 🎙️ Supported Commands

| Command                           | Action               |
| --------------------------------- | -------------------- |
| "Hello Jarvis"                    | Activates Jarvis     |
| "Wake Up"                         | Starts listening     |
| "Open Google"                     | Opens Google         |
| "Open YouTube"                    | Opens YouTube        |
| "Open GitHub"                     | Opens GitHub         |
| "Search for AI tools"             | Searches Google      |
| "Play Imagine Dragons on YouTube" | Opens YouTube search |
| "What time is it?"                | Reads current time   |
| "What's today's date?"            | Reads current date   |
| "Goodbye Jarvis"                  | Stops listening      |
| "Sleep"                           | Ends session         |

## 📦 Installation

### Clone the Repository

```bash
git clone https://github.com/abuhuzaifa12332-lab/jarvis-ai.git
```

### Navigate to Project Directory

```bash
cd jarvis-ai
```

### Run Local Server

```bash
python3 -m http.server 8080
```

### Open Browser

```text
http://localhost:8080
```

## ⚠️ Browser Requirements

For the best experience:

* Allow microphone access when prompted.
* Use Google Chrome (Recommended).
* Microsoft Edge is also supported.

## 🔒 Privacy

Jarvis AI operates entirely inside the browser. Voice commands are processed using browser-provided speech technologies, and no custom backend server is required for operation.
